Panel en preparación

El panel administrativo se activa al conectar Sanity.

La web ya está lista. Para que tu papá pueda editar textos e imágenes desde un panel simple, solo falta conectar el proyecto de Sanity y volver a desplegar.

  1. Crear el proyecto gratuito en Sanity.
  2. Configurar NEXT_PUBLIC_SANITY_PROJECT_ID y NEXT_PUBLIC_SANITY_DATASET.
  3. Configurar SANITY_STUDIO_PROJECT_ID y SANITY_STUDIO_DATASET en GitHub Actions.
  4. Volver a desplegar para publicar el panel real en esta misma URL.

Mientras tanto, consulta la guía del proyecto en el sitio principal.
